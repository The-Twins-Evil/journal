## Journal
Online daily mindfulness journal

## Server
For testing, using json-server.
This will be replaced with either local storage or a database before production.

## Usage

To build bundle, in terminal: "npm run build"


## Requirements

- React 17.0.1
- ReactDOM 17.0.1
- React Router 5.2.0
- Babel 6.23.0
- CSS Loader 5.0.1
- Style Loader 2.0.0
- Webpack 4.44.2
- Webpack Cli 3.3.12
